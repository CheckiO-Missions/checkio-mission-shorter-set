Ідея, яку я тобі пропоную, це правильно "відрізати" зайве від сортованого списку.
З якого засобу Python мені розпочати для втілення цієї ідеї в код?
Оскільки множина це невпорядкований набір, тобі потрібно створити новий впорядкований набір, який можна буде відсортувати. Список підійде для цього. Тож, спершу тобі потрібна функція sorted() для створення сортованого списку з вхідної множини. 
data = {1, 5, 2, 4}
order = sorted(data)
print(order) # [1, 2, 4, 5] ти отримав список,  відсортований по зростанню.
І як тепер видалити певну кількість мінімальних і максимальних значень з цього списку?
Твій список відсортований по зростанню, тобто він починається із найменших значень і закінчується найбільшими. Тепер ти можеш використати зрізування з позитивними і негативними індексами для отримання нового часткового списку. 
order_sl = order[1:-1] # ти отримав новий список з попереднього
                    # зі зрізаними першим і -першим(останнім) значеннями.
print(order_sl) # [2, 4]
Добре. Але досі не розумію, які індекси мені використати для зрізу в моєму розв'язку.
Ти маєш вхідне значення total. Воно показує скільки найменших і найбільших значень має бути видалено із множини. Використай його як позитивний і негативний індекс, щоб зрізати цю кількість значень зі свого списку. 
order = sorted(data)
order_sl = order[total:-total]
Після попереднього кроку я маю список, але ж розвязок має повертати множину. Щоб мені слід зробити для цього?
Так, тобі потрібно лише створити нову множину зі свого списку з допомогою set(order_sl) і повернути її. 
order = sorted(data)
order_sl = order[total:-total]

return set(order_sl)
І останнє питання. Мій зріз не працює, якщо total == 0. Як включити цей варіант у розв'язок?
Ти маєш перевірити значення total перед іншими командами, щоб уникнути зайвих розрахунків. Ти можеш зробити це, написавши 'if total == 0:', але 'pythonic' спосіб це зробити, це написати лише 'if not total:'. Оскільки нічого видаляти не потрібно, ти повертаєш початкову множину. І не забудь проголосити саму функцію. 
def remove_min_max(data: set[int], total:int) -> set[int]:
    
    if not total:
        return data

    order = sorted(data)
    order_sl = order[total:-total]

    return set(order_sl)
